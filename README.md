# eZ80-IDE-Festplatte

## Entwicklung eines modularen Z80- und eZ80-Computersystems

Dieses Repository dokumentiert die Entwicklung eines modularen
Computersystems auf Basis der Zilog-Z80-Prozessorfamilie.

Im Mittelpunkt stehen die Entwicklung eines eigenen BIOS, die
Anbindung von IDE-Festplatten und CompactFlash-Medien sowie die
Unterstützung weiterer Massenspeicher wie MMC- und SD-Karten.

Das Projekt begann mit einem selbst entwickelten Z80-System und
wird heute mit dem Ziel weitergeführt, die Software auf den eZ80
zu portieren und die Entwicklung vollständig zu dokumentieren.

## Projektziele

- Entwicklung eines modularen BIOS
- Hardwareunabhängige Software durch konfigurierbare Include-Dateien
- Unterstützung verschiedener Massenspeicher
- Dokumentation der Hardware und Software
- Portierung auf den eZ80

## Projektstatus

🚧 Das Projekt befindet sich in aktiver Entwicklung.

Die Software wird kontinuierlich erweitert und getestet.
Schaltpläne und Hardwareunterlagen werden nach erfolgreichem
Abschluss der jeweiligen Entwicklungsphase veröffentlicht.

## Verzeichnisstruktur
