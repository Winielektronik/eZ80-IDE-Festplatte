
# Dokumentation

Dieses Verzeichnis enthält die technische Dokumentation des Projekts
**eZ80-IDE-Festplatte**.

## Inhalt

Hier werden nach und nach die einzelnen Komponenten des Systems beschrieben:

- BIOS
- Bootvorgang
- Speicherverwaltung
- Bankumschaltung
- IDE-Festplatteninterface
- MMC-/SD-Karteninterface
- Ein-/Ausgabe (SIO, PIO, CTC)
- LCD-Ansteuerung
- Hardwareplattformen
- eZ80-Portierung

## Ziel

Die Dokumentation soll den Aufbau und die Funktionsweise der Hard-
und Software verständlich beschreiben und die Weiterentwicklung
des Projekts unterstützen.

Die Dokumentation wird parallel zur Entwicklung kontinuierlich erweitert.
